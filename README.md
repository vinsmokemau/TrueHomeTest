# TrueHomeTest
This repository contains a backend test for TrueHome

## Run project


```sh
docker-compose -f local.yml build
docker-compose -f local.yml up
```

## Ports
Django - 8000
Vue - 8080

## Django endpoints
Activity endpoints - activities/

## Vue urls
Activities - /
